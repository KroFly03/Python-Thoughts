Примеры команд системы контроля версий, которые я использую:

git clone - Клонирует репозиторий на локальную машину
git init - Инициализирует новый Git-репозиторий
git add - Добавляет файлы в индекс для последующего коммита
git commit -m "text" - Фиксирует изменения в репозитории с сообщением коммита.
git pull - Получает изменения с удаленного репозитория
git push - Отправляет изменения в удаленный репозиторий
git branch - Показывает список локальных веток
git checkout - Переключается на указанную ветку
git merge - Сливает указанную ветку с текущей, оставляя коммит об слиянии в текущей ветке
git status - Показывает состояние рабочего каталога
git log - Показывает историю коммитов
git remote -v - Показывает информацию о удаленных репозиториях
git reset --hard HEAD - Сбрасывает текущие изменения в последний коммит
git restore --staged - Отменяет индексацию изменения для файла
git rebase - Сливает указанную ветку с текущей, оставляя текущую после последнего коммита из основной ветки
